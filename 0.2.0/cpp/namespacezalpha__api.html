bool zalpha_api::versionCompatible ( std::string  api_server_version)

Checks whether the API server version is compatilble with the client library version.

The version number consists of three parts: {major}.{minor}.{patch}. The set of API commands will be similar and compatible as long as the major and minor version numbers remain the same.

Parameters
api_server_versionThe version string of the API server.
Returns
A boolean indicating whether this client libary version is compatible.
